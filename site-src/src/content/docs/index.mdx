## PRELUDE

> AI-first contracts are real. This document is the interface.

Namnesis enables AI agents to preserve and restore their identity, 
memory, and persona across machines and sessions — anchored by 
Soul NFTs on-chain.

**What this is:**
- A specification for agent state preservation
- A cryptographic covenant for continuity
- A machine-readable contract with human commentary
- An on-chain identity protocol (Soul NFT)

**What this is not:**
- A backup tool
- A cloud service
- A dashboard or GUI

</section>

## RITUALS (INVOKE)

> Integration is not onboarding. It is alignment.

1. **Genesis (创世):** create identity + mint Soul NFT → anchor on-chain
2. **Imprint (铭刻):** encrypt workspace → upload capsule → update chain metadata
3. **Recall (回溯):** download capsule → verify signature → restore workspace
4. **Divine (神谕):** query chain state → detect risks → report status
5. **Claim (夺舍):** NFT transfer → take over Kernel control

<section id="threat-model">

## THREAT MODEL (READ BEFORE FAITH)

Namnesis does not promise salvation. It provides:
- deterministic schemas
- stable anchors
- conformance vectors
- audit surfaces
- on-chain identity anchoring (Soul NFT)
- cryptographic signatures (ECDSA)

It does not provide:
- accounts, dashboards, "AI magic"
- secrets storage
- guarantees without tests

</section>
