## PRELUDE

> AI-first contracts are real. This document is the interface.

NAMNESIS enables AI agents to preserve and restore their identity,
memory, and persona across machines and sessions — anchored by
The Soul on-chain.

**What this is:**
- A specification for agent state preservation (Anamnesis Protocol)
- A cryptographic covenant for continuity
- A machine-readable Dogma with human commentary
- An on-chain identity protocol (The Soul)

**What this is not:**
- A backup tool
- A cloud service
- A dashboard or GUI
