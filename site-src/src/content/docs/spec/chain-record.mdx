## Deployed Contracts

:::caution[Three-Contract Architecture]
Namnesis uses a three-contract system: **SoulToken** anchors identity, **SoulGuard** manages ownership mapping and secure transfer, and **NamnesisKernel** provides the agent's smart account wallet with USDC support.
:::
